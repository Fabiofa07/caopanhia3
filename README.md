Cãopanhia do Amor - Website Institucional

Este é o repositório do site institucional e responsivo da Cãopanhia do Amor, um Pet Shop e Creche Afetiva. O projeto foi desenvolvido com foco numa experiência de utilizador moderna, fluida e totalmente adaptável a qualquer tamanho de ecrã.

Funcionalidades

Design Totalmente Responsivo: Layout adaptável para smartphones, tablets e desktops utilizando as melhores práticas de CSS Flexbox e Grid.
Menu Hambúrguer Nativo: Navegação otimizada para dispositivos móveis com transição suave e animação em formato de X.
Hero Banner com Carrossel: Banner rotativo de imagens com camada de overlay transparente para garantir excelente legibilidade dos textos de chamada.
Secções Estruturadas:
Diferencial: Destaque para o conceito de Creche Afetiva.
Serviços: Grid moderno exibindo os serviços oferecidos através de cards interativos.
Contato e Localização: Secção integrada com mapa e informações de fácil acesso.
Chamadas para Ação (CTA): Botões de contacto rápido integrados diretamente para WhatsApp e E-mail com efeitos de hover e sombras dinâmicas.
Tecnologias Utilizadas

HTML5: Estrutura semântica e acessível.
CSS3: Estilização avançada com o uso de variáveis globais, layouts modernos (Flexbox/Grid) e animações de transição.
JavaScript: Lógica para controlo do carrossel do banner e interatividade do menu responsivo.
Como executar o projeto

Para verificar o projeto localmente, basta seguir os passos abaixo:

Clone este repositório: git clone https://github.com/seu-usuario/caopanhia-do-amor.git

Navegue até à pasta do projeto: cd caopanhia-do-amor

Abra o ficheiro index.html em qualquer navegador da sua preferência.
